# Build from source

out-of-tree build and out-of-tree install.

```
$ (sudo apt-get build-dep zsh)
$ ./Util/preconfig
$ mkdir -p out/Default
$ cd out/Default
$ mkdir -p _install
$ ../../configure --prefix=_install
$ make -j 2
$ make install
$ ./_install/bin/zsh
```
