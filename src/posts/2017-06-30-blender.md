# Build system

Build from source:

```
$ mkdir out/Default
$ cd out/Default
$ mkdir -p _install
$ cmake -G Ninja -DPYTHON_VERSION=3.6 -DCMAKE_BUILD_TYPE=Debug -DCMAKE_INSTALL_PREFIX=$PWD/_install ../..
$ ninja -t browse # check build artifacts dependency for feeling overview
$ ninja install
$ ./_install/blender
```
